# 工作岗位评价知识库

本知识库按**岗位**维护：每个岗位在通用能力之外，列出**差异化（特殊）评价指标**，便于实习鉴定、周报反馈、转正答辩或与 JD 对齐。

**当前范围（刻意收窄）**：**计算机类**（软件开发、数据/算法/AI、质量与基础设施）+ **人力与会计学**（HR/SSC、财务会计、FP&A、审计）。各文件内指标已**加细**，强调可观察行为、证据链与及格线/加分项。

## 与姊妹文档的关系

岗位名称与场景的粗分类见上级目录：`工科与管理学-实习就业岗位参考.md`。  
本目录在「有哪些岗位」之上，聚焦「**该岗位靠什么指标区分好与一般**」。

## 维护方式建议

1. **新增岗位**：从 `00-通用底座与条目模板.md` 复制模板一节，放到对应分类文件末尾。  
2. **特殊指标**：只写该岗位**相对其他岗位更关键**的观察点；通用项（态度、沟通等）不重复展开。  
3. **版本**：指标随技术栈与业务变，建议在每个文件底部用一行记录「最近梳理日期」。
